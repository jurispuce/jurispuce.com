## Module Overview

The capstone project brings together everything you've learned throughout the AI Security Mastery path. You'll work on a realistic AI system, applying both technical and governance skills.

## Project Scope

You will:
1. Conduct a complete security assessment of an AI system
2. Identify and prioritize vulnerabilities
3. Design a governance framework
4. Implement security controls
5. Document your findings and recommendations
6. Present your work for evaluation

## Evaluation Criteria

- Technical depth and accuracy
- Comprehensive risk assessment
- Practical and implementable recommendations
- Quality of documentation
- Presentation and communication skills
