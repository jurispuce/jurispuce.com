Standarta platformas

Gatavas komerciālas programmatūras no pazīstamiem piegādātājiem, kas netiek būtiski modificētas

Piemēri:

  • Microsoft Office 365 - productivity suite
  • Windows Server & Desktop - operētājsistēmas
  • Active Directory - autentifikācijas sistēma
  • SharePoint - sadarbības platforma
🔧

Pielāgojamas platformas

Komerciālas programmatūras ar plašām konfigurācijas iespējām un customization funkcionalitāti

Piemēri:

  • VISMA Horizon ERP - grāmatvedības un finanšu sistēma
  • Namejs DMS - dokumentu pārvaldības sistēma
  • DocLogix - lietvedības sistēma
  • MicroStrategy - biznesa analīzes platforma
⚡

Pielāgotas platformas

Unikāli izstrādātas informācijas sistēmas specifiskiem tieslietu sektora procesiem

Piemēri:

  • TIS (Tiesu informācijas sistēma) - lietu vadības sistēma
  • E-lietas ekosistēma - izmeklēšanas un tiesvedības process
  • VZD kadastra sistēmas - nekustamā īpašuma reģistri
  • Uzņēmumu reģistra sistēmas - juridisko personu dati
